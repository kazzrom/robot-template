### Создание структуры робота
Для удобства создания проекта робота используется менеджер зависимостей `uv`.
Для создания проекта перейдите в папку, в которой хотите создать проект и запустите следующую команду:
```bash
uvx copier copy "https://github.com/kazzrom/robot-template.git" . --trust
```
`copier copy` принимает два пути, в случае примера:
- `"https://github.com/kazzrom/robot-template.git"` - репозиторий-шаблон
- `.` - путь, где будет инициализирован проект. В данном случае в корне директории, в которой запускается скрипт.

### Опции конфигурации ([подробнее](docs/OPTIONS.md))
При запуске шаблонизатор предложит выбрать параметры проекта:
| Опция | Описание |
| :--- | :--- |
| `include_it_enterprise` | Включить файлы для работы с IT-Enterprise? |
| `install_playwright` | Установить Playwright? |
| `use_dev_containers` | Использовать Dev Containers? |

Сгенерированный проект на основе этого шаблоны придерживается стандартов [rf-pw-standards](https://github.com/kazzrom/rf-pw-standards)
